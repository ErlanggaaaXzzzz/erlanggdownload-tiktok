Pakai API-nya di project kamu sendiri

Satu endpoint REST, balikin JSON berisi link video HD, versi watermark, dan audio. Butuh API key — generate gratis di bawah, lalu kirim lewat header atau query string.

Bikin & cek API key lewat endpoint

Tombol "Generate API key" di atas manggil endpoint ini juga. Key yang dibuat tersimpan di server, bukan cuma di browser kamu.

POST /api/keys — buat key baru 
curl -X POST "https://erlanggadownloadtiktok.vercel.app/api/keys"

// response
{
  "success": true,
  "data": {
    "api_key": "ERL-3f9a8b2c1d",
    "created_at": "2026-07-21T10:00:00.000Z"
  }
}
GET /api/keys?key=... — cek status key 
curl "https://erlanggadownloadtiktok.vercel.app/api/keys?key=ERL-3f9a8b2c1d"

// response
{
  "success": true,
  "data": {
    "api_key": "ERL-3f9a8b2c1d",
    "active": true,
    "request_count": 12
  }
}
